# MyFitBot - Навык фитнес-трекера для Алисы 🏃‍♀️🏃‍♂️

**MyFitBot** — это бэкенд для навыка Яндекс.Алисы, написанный на Python с использованием фреймворка Flask. Навык позволяет вести голосовой дневник пробежек, запрашивать статистику и узнавать погоду перед тренировкой. Также предусмотрен веб-интерфейс для загрузки медиафайлов.

Проект создан для демонстрации работы с Webhook Алисы, базами данных (ORM) и сторонними API.

---

## 📋 Описание

MyFitBot выступает связующим звеном между пользователем (через интерфейс Алисы) и базой данных. Вы просто говорите колонке или телефону: *"Алиса, попроси фитнес-трекер записать, что я пробежал 5 километров"*, и приложение автоматически распознает цифру с помощью NLU-сущностей Яндекса и сохраняет ее в базу данных. 

## 🎯 Возможности

* **Голосовой ввод тренировок** — автоматическое извлечение чисел (дистанции) из речи.
* **Генерация отчетов** — подсчет общей дистанции по запросу "Отчет".
* **Интеграция стороннего API** — прогноз погоды по запросу "Погода в [Город]" (через OpenWeatherMap).
* **Контекст пользователя** — распознавание уникальных пользователей через `user_id` сессии Алисы.
* **Веб-интерфейс** — простая HTML-форма для загрузки фотографий с тренировок или плейлистов музыки на сервер.
* **ORM База данных** — использование SQLAlchemy и SQLite для надежного хранения истории.

## 💻 Требования

* Python 3.8 или новее
* Операционная система: Windows 10/11, macOS, Linux
* Учетная запись разработчика Яндекс.Диалогов
* (Опционально) API-ключ от OpenWeatherMap
