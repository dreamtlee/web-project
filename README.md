# 🏋️ MyFitBot Pro — Интеллектуальный Фитнес-Аналитик для Алисы

**MyFitBot** — это продвинутая экосистема для Яндекс.Алисы, сочетающая в себе голосового помощника, аналитический модуль и веб-дашборд. Бэкенд реализован на **Python (Flask)** с использованием **SQLAlchemy ORM**.

Проект перерос рамки простого логгера и теперь выполняет роль персонального тренера: рассчитывает показатели здоровья, отслеживает прогресс и мотивирует пользователя.

---

## 🎯 Основные возможности

### 🤖 Голосовой интерфейс (Яндекс.Алиса)
* **Интеллектуальный логгер:** Автоматическое распознавание дистанции через NLU-сущности `YANDEX.NUMBER`.
* **Персональное профилирование:** Настройка веса и роста голосом для точных расчетов.
* **Погодный модуль:** Интеграция с **Yandex Weather API** и **Geocoder API** для планирования тренировок.
* **Мотивация:** Система случайных фитнес-советов при каждом запуске навыка.

### 📊 Аналитический модуль (Pro функции)
* **Расчет ИМТ:** Автоматическое вычисление индекса массы тела по формуле:
  $$BMI = \frac{weight}{height^2}$$
* **Система уровней:** Игровая механика достижений (Новичок -> Атлет -> Марафонец).
* **Deep Analytics:** Расчет среднего темпа (мин/км), поиск лучшей тренировки и прогноз достижения цели.

### 🌐 Веб-интерфейс (Dashboard)
* **Визуализация:** Интерактивная таблица последних тренировок.
* **Cloud Storage:** Загрузка и хранение фотографий или аудиофайлов в папку `static/uploads`.
* **Глобальная статистика:** Мониторинг общего количества пользователей и суммарного пробега системы.

## 💻 Технический стек

* **Ядро:** Python 3.10+
* **Framework:** Flask (Web + Webhook)
* **Database:** SQLite + SQLAlchemy (ORM-подход)
* **API:** Яндекс.Диалоги, Яндекс.Погода, Яндекс.Геокодер
* **Frontend:** Jinja2, HTML5, CSS3 (Адаптивный дизайн)

## 🏆 Система достижений

Приложение автоматически присваивает статусы на основе общего прогресса:

* **Новичок:** до 10 км.
* **Атлет:** от 10 до 50 км.
* **Марафонец:** от 50 до 150 км.
* **Ультрамарафонец:** свыше 150 км.
